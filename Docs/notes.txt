Eelsõna:
Kuna lahendus peab olema käivitatav Windowsi op-süsteemiga, siis enda jaoks proovisin mitte-igapäevase arvutiga.

1) Paigaldasin eeldused:
    1. Terraform
    2. Lisasin PATHi terraformi
    3. Podman Desktop
    4. Ansible
        * Kasutasin Cygwin, alternatiiv oleks WSL - Podman Desktop installides sain aru, et ümber selle ei saa
        * Sain ka aru, et Ansiblet pole "host" masinal vaja



Probleemkohad / tähelepanekud:
    1) Terraform apply ei suuda normaalselt hävitada podmani konteinereid - proovisin eri workarounde (nt force, remove volumes jms)
    2) Ansible jooksutamist ei saanud korralikult tööle - minu lahendus oli,
        et TF loob konteineri, mille sees ansible jookseb, aga mingi nõks jäi puudu
        * Error: Unable to read Docker image into resource: unable to list Docker images: Get "http://%2F%2F.%2F%2Fpipe%2F%2Fpodman-machine-default/v1.41/images/json": context deadline exceeded
    3) Windows powershelli peal on väga harjumatu käske jooksutada
    4) Podman tundub päris kiire. (Olen varem kasutanud kind / docker / k8s)
